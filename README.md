# 🌱 My Spring Boot Journey

Welcome to my personal learning repository! 

This repo is dedicated to documenting my progress, experiments, and projects as I explore **Spring Boot** and the broader Spring ecosystem.

## 🚀 Goals
- Build a strong foundation in Spring Boot and Spring Core.
- Create REST APIs and understand how to structure backend applications.
- Learn about data persistence with Spring Data JPA and Hibernate.
- Explore Spring Security for authentication and authorization.
- Deploy applications using Docker and cloud platforms.
- Share resources, and mini-projects to help others on the same path.
